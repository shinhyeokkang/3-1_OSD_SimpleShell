How to turn on my Program
1. Download readme.txt , SimpleShell.java 
2. open terminal in directory which contains SimpleShell.java
3. type "javac SimpleShell.java"
4. type "java SimpleShell"
5. then my program will run.
-------------------------------------------------------
How to use my Shell
1. type 'exit' or 'quit' to quit.
2. type ps, ls, and cat command just like in terminal
3. type pwd to see current directory
4. type cd to change directory to first place
5. type cd /home to change directory to /home
6. type cd 'yourusername' to change directory to /home/user
7. type cd/'user'/'existingfolder' to change directory to /home/user/folder
8. type cd .. to change directory to go back once
9. type cd fakeDirectory if you want to check errer message
10. type history to see all history
11. type history# (#=any number in history) to see specific history
12. type !! to run last command again
13. type !# (#=any number in history) to run specific command in history
